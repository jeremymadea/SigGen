# SigGen

A sine and square wave generator based on Arduino and an AD9850 DDS module.

## Parts

* Arduino 
* 2x16 LCD
* I2C backpack for the LCD
* AD9850 Module 
* Rotary Encoder 
* Push button switch (normally open) 
* 100 Ohm resistor
* Rocker switch (SPST) 
* BNC comnector 
* Enclosure 

## Dependencies 

Rotary encoder Arduino library
